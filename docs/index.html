Key Financial Formulas

Net Present Value (NPV):

\( \mathrm{NPV} = \sum_{t=0}^{T} \dfrac{C_t}{(1 + r)^t} \)

Compound Annual Growth Rate (CAGR):

\( \mathrm{CAGR} = \left(\dfrac{V_{f}}{V_{i}}\right)^{\frac{1}{n}} - 1 \)

Small code: NPV calculator (Python)


def npv(cashflows, discount_rate):
    """
    cashflows: list of floats, where cashflows[0] is C0 (initial)
    discount_rate: decimal (e.g., 0.08)
    """
    return sum(cf / ((1 + discount_rate) ** t) for t, cf in enumerate(cashflows))

# Example:
cashflows = [-1000, 400, 400, 400]
print(npv(cashflows, 0.08))
Mention assumptions: timing of cashflows, terminal value, discount rate selection.
